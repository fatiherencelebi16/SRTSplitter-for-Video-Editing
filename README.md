# Modern SRT Splitter

Modern SRT Splitter is a Windows application that splits SRT subtitle files into **2-word chunks (or any number of words you choose)**.

---

## Features
- Standalone EXE (No Python required)  
- Adjustable chunk size using a slider  
- Batch file processing supported  
- Drag & drop or file selection for easy use  
- Color-coded log for process tracking  
- Full Turkish character support  

---

## Usage
1. Open the EXE file.  
2. Click `Add SRT Files` or drag & drop subtitle files.  
3. Choose an output folder using `Select Output Folder`.  
4. Adjust the chunk size with the slider (default 2).  
5. Click `Start Processing` to begin.  
6. Find new SRT files in the output folder once done.  

---

## Tips
- Use UTF-8 encoded SRT files.  
- Turkish characters can be used in file and folder names.  
- The EXE is portable, no installation needed.

# Modern SRT Splitter (Türkçe)

Modern SRT Splitter, SRT altyazı dosyalarını **2’şer kelime (veya istediğiniz sayıda)** olacak şekilde bölen Windows uygulamasıdır.

---

## Özellikler
- Tek dosya çalıştırılabilir EXE (Python gerektirmez)  
- Chunk size slider ile kelime sayısı ayarlanabilir  
- Toplu dosya işleme destekli  
- Sürükle-bırak veya dosya seçme ile kolay kullanım  
- Renkli log alanı ile işlem durumu takip edilebilir  
- Türkçe karakter desteği  

---

## Kullanım
1. EXE dosyasını açın.  
2. `Add SRT Files` butonuyla altyazı dosyalarını seçin veya sürükleyin.  
3. `Select Output Folder` ile çıktı klasörünü seçin.  
4. `Chunk size` slider ile her satırdaki kelime sayısını belirleyin (varsayılan 2).  
5. `Start Processing` ile işlemi başlatın.  
6. İşlem tamamlandığında seçtiğiniz klasörde yeni SRT dosyaları oluşacaktır.  

---

## Öneriler
- UTF-8 kodlamalı SRT dosyaları kullanın.  
- Dosya adı ve klasör isimlerinde Türkçe karakter kullanabilirsiniz.  
- EXE dosyasını taşınabilir şekilde kullanabilirsiniz, kurulum gerekmez.
